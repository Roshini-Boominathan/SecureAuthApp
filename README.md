# SecureAuthApp

## Description
SecureAuthApp is a Java-based authentication mini project developed as part of a Full Stack Development Internship.  
It provides user registration, login, logout, and session-based authentication using JSP, Servlets, and JDBC.

## Technologies Used
- Java (JDK 21)
- JSP & Servlets
- JDBC
- MySQL
- Apache Tomcat 10
- Eclipse IDE

## Features
- User registration
- Secure login authentication
- Session management
- Logout functionality
- Simple dashboard page

## Project Structure
src/main/java/com/auth/
├── controller
├── dao
├── model
└── util

src/main/webapp/
├── index.jsp
├── login.jsp
├── register.jsp
├── dashboard.jsp

## How to Run
1. Import the project into Eclipse
2. Configure MySQL database and update credentials in `DBUtil.java`
3. Deploy the project on Apache Tomcat
4. Access the application via browser

## Internship Task
This project is developed as Task 1 for a Full Stack Development Internship.
